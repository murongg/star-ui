<center>Star UI - A Vue.js 3.0 Mobile UI library</center>

## Install

```
npm i star-ui -S
```

## Quickstart

```js
import { createApp } from 'vue';
import { Icon } from 'star-ui';

const app = createApp();
app.use(Icon);
```
