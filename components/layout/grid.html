# Grid 宫格

# 介绍

基于 CSS Grid 的宫格组件

# 引入

import { createApp } from 'vue'
import { Grid, GridItem } from 'star-ui';
import App from './App.vue';

createApp(App).use(Grid).use(GridItem)

# 基本使用

通过设置 Grid 组件 cols 属性来确定宫格所占列数，默认值为 3。

<star-grid :cols="3">
  <star-grid-item>
    <div class="block">
      <star-icon name="delete"></star-icon>
      图标
    </div>
  </star-grid-item>
 ...
</star-grid>

# 宫格间隔

通过设置 Grid 组件 x-gap、y-gap 属性修改宫格间隔，两者默认值均为 0。

  • x-gap：横向间隔槽
  • y-gap：纵向间隔槽
<star-grid :x-gap="10" :y-gap="10">
  <star-grid-item>
    <div class="block">
      <star-icon name="delete"></star-icon>
      图标
    </div>
  </star-grid-item>
  ...
</star-grid>

# 宫格偏移

通过设置 GridItem 组件 offset 属性设置宫格向左偏移，默认值为 0。

<star-grid :cols="5">
  <star-grid-item :offset="1">
    <div class="block">
      <star-icon name="delete"></star-icon>
      图标
    </div>
  </star-grid-item>
  <star-grid-item :offset="1">
    <div class="block">
      <star-icon name="delete"></star-icon>
      图标
    </div>
  </star-grid-item>
</star-grid>

# 宫格所占列数

通过设置 GridItem 组件 span 属性设置宫格所占列数，默认值为 1。

<star-grid :cols="5">
  <star-grid-item :span="2">
    <div class="block">
      <star-icon name="delete"></star-icon>

      图标

# API

# Grid Props

参数说明类型可选值默认值
cols每行显示的宫格数目Number3
x-gap横向间隔槽，默认单位为 pxNumber0
y-gap纵向间隔槽，默认单位为 pxNumber0
border是否显示所有的边框Booleanfalse
center是否将格子内容居中显示Booleantrue

# Grid Item Props

参数说明类型可选值默认值
span宫格所占格数Number1
offset宫格左侧的间隔格数Number0
