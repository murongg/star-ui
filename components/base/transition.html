# Transition 过渡

# 介绍

常用的过渡动画。
此组件引入了 animation.css 的常用动画

# 引入

import { createApp } from 'vue'
import { Transition } from 'star-ui';
import App from './App.vue';

createApp(App).use(Transition)

# 基础使用

通过设置 name 属性修改动画类型。

<template>
  <star-button size="large" shape="semicircle" @click="handleClickShow">切换显示/隐藏</star-button>
  <star-transition name="fade" v-model="show">
    <div class="block"></div>
  </star-transition>
</template>

<script>
export default {
  data() {
    return {
      show: true,
    };
  },
  methods: {
    handleClickShow() {
      this.show = !this.show;
    },
  },
};
</script>

# 动画时长

通过设置 duration 属性修改动画时长，单位 ms。

<star-transition name="fade" :duration="500">
  <div class="block" v-show="show"></div>
</star-transition>

# API

# Props

参数说明类型可选值默认值
model-value / v-model动画显示/隐藏Booleanfalse
name动画类型Stringfade
duration动画时长String、Number500

# 动画列表

# Back

名称说明
back-down从下方退出
back-up从上方退出
back-left从左侧退出
back-right从右侧退出

# Fading

名称说明
fade淡入淡出
fade-down从下方淡入淡出
fade-up从上方淡入淡出
fade-left从左侧淡入淡出
fade-right从右侧淡入淡出

# Sliding

名称说明
slide-down从下方滑出
slide-up从上方滑出
slide-left从左侧滑出
slide-right从右侧滑出

# Zooming

名称说明
zoom变焦
zoom-down从下方变焦出
zoom-up从上方变焦出
zoom-left从左侧变焦出
zoom-right从右侧变焦出

# Bouncing

名称说明
bounce跳跃
bounce-down从下方跳出
bounce-up从上方跳出
bounce-left从左侧跳出
bounce-right从右侧跳出

# Rotating

名称说明

rotate-down-right从右下方旋转出
rotate-up-left从左上方旋转出
